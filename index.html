Tips (SoldF 2001)

Ange tidsnummer — sex siffror: dag, timme, minut. Onsdag 4 mars kl 21:15 = 042115. Ange alltid observationstidpunkten, inte när du rapporterar. Vid andrahandsuppgift: ange tydligt att tiden är osäker.

📍 = din position  ·  🗺️ = välj observationsposition på karta

Tips (SoldF 2001)

Militärt standard är MGRS. Ange alltid vilket koordinatsystem du använder om det kan råda tveksamhet. Använd aldrig UPK (utgångspunkter på kartan) för att ange motståndarens position — det kan röja det egna UPK-systemet.

Tips (SoldF 2001)

Sträva efter exakt antal. Ange "minst X" om du misstänker att det är fler än du kan bekräfta. Vid fotburet infanteri i terräng kan du ange "bedömt i plutons storlek" för att indikera storlek utan att ge sken av exakthet.

Tips (SoldF 2001)

Ange slag så detaljerat som möjligt. För fordon: hjul eller band, antal axlar/bärhjul, torn (stort/litet), beväpning, om det bär trupp, bepansring, synliga skador. Om du inte kan identifiera fordonstypen exakt — beskriv vad du ser. Gissa inte vad du inte är säker på.

Tips (SoldF 2001)

Var dom under transport, stod still och väntade, fyllde på förbrukning (ammunition/drivmedel), genomförde reparationer, eller förberedde försvar? All information bidrar till att underrättelsebefälet förstår vad som är i görningen.

Tips (SoldF 2001)

Symboler hjälper underrättelsepersonal att skilja förband åt och identifiera rörelsemönster. Rapportera allt unikt: flaggor, markeringar, registreringsnummer, avvikande utrustning, synliga skador. Samma skada på fordon i flera rapporter kan bekräfta att det är samma enhet.

Tips (SoldF 2001)

Ange individ eller enhetsbeteckning. Uppgiften är viktig om det finns följdfrågor — t.ex. för att validera tolkningar eller koppla rapporten till andra observationer. Tydlig sagesman möjliggör att staben kan kontakta den som gjort observationen.

Tips (SoldF 2001)

Vad rapporterande enhet gör efter observationen. Viktigt för överordnad chef och underrättelsebefäl — om enheten inte hör av sig igen ger "Sedan" vägledning om vad som kan ha skett och var.
